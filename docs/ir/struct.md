# Локальные структуры — коды 0x50–0x57

| Код (hex) | Команда  | Параметры                      | Описание                                 |
|-----------|----------|--------------------------------|------------------------------------------|
| 0x50      | store_at | base_offset, field_offset, src | Записать в локальный blob по смещению    |
| 0x51      | load_at  | base_offset, field_offset, dst | Прочитать из локального blob по смещению |
| 0x52–0x57 | —        | —                              | (резерв)                                 |
