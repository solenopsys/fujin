# 2. Операторы

## Арифметические

```fujin
+ - * / %        // базовые
++ --            // инкремент/декремент
```

## Логические

```fujin
&& || !          // AND, OR, NOT
```

## Сравнение

```fujin
===  !==         // строгое равенство (только они!)
<  <=  >  >=     // сравнения
```

## Присваивание

```fujin
=                // простое
+= -= *= /= %=   // арифметическое
```

Прочие операторы (`??`, тернарный, побитовые, delete/void/typeof/in/instanceof, логическое присваивание) отсутствуют в языке.
